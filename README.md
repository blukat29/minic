Mini-C compiler
===============

KAIST CS420 Introduction to Compiler Project

### Requirements
- java
- javac
- make
- wget
- tar

### Setup
```
make install
make
make test
```

### Run

```
make run < input_file.c
```
or
```
./mcc < input_file.c
./mcc input_file.c
```

### Based on
- JFlex
- CUP
